```css
/* CSS-Only 오버레이로 구조 변환 */
.container {
    display: grid !important;
    grid-template-columns: 500px 1fr !important;
    gap: 30px !important;
}
```
- **핵심**: 수식 기반 계산 로직을 한 글자도 수정하지 않음
