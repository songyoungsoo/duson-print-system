# 6섹션 전단지 구조 — 소규모 사업체 홍보 특화

> 이 문서는 카피라이팅/디자인 에이전트의 시스템 프롬프트에 포함됩니다.
> 각 섹션의 목적, 구성 요소, 심리적 트리거를 정의합니다.
> A4 양면 전단지(794x1123px x 6섹션)에 최적화되어 있습니다.

---

## 전체 흐름 (AIDA 심리 퍼널)

```
┌─────────────────────────────────────────────────────────────┐
│  앞면 (Front) — 3초 안에 관심 끌고 행동 유도                  │
│                                                               │
│  Section 1: hero ──────── Attention (주의 끌기)               │
│  Section 2: menu ──────── Interest (관심 유발)                │
│  Section 3: promo_contact  Action (행동 유도)                 │
│                                                               │
├─────────────────────────────────────────────────────────────┤
│  뒷면 (Back) — 신뢰 형성하고 방문 유도                        │
│                                                               │
│  Section 4: features ──── Trust (신뢰 확보)                   │
│  Section 5: gallery ───── Desire (욕구 자극)                  │
│  Section 6: location ──── Navigate (길 안내)                  │
│                                                               │
└─────────────────────────────────────────────────────────────┘
```

### 심리 흐름 설계 의도
```
앞면: "뭐지?" → "오, 이런 메뉴가!" → "전화해야지/가봐야지"
뒷면: "이 가게 뭐가 좋지?" → "분위기 좋네" → "여기가 어디지? 찾아가자"
```

> **핵심 원칙**: 전단지는 3초 안에 승부난다. 앞면만 보고 버릴지/뒤집을지 결정된다.
> 따라서 앞면 3섹션이 가장 중요하고, 뒷면은 "뒤집은 사람"을 위한 보충 정보다.

## 인쇄물 타이포그래피/레이아웃 가이드라인

### A4 인쇄 기본 규격
```
전단지 크기: A4 (210x297mm) → 794x1123px (300dpi 기준 리사이즈)
재단 여백: 각 변 3mm (안전 영역 안에 핵심 정보 배치)
접지선: 없음 (낱장 단면+양면)
```

### 가독성 규칙
```
1. 최소 폰트: 인쇄물에서 8pt 이상 (화면에서 ~14px 상당)
2. 행간(line-height): 본문 1.6배, 헤드라인 1.2배
3. 자간(letter-spacing): 한글 0, 영문 약간 좁게
4. 대비: 배경 대비 최소 4.5:1 (WCAG AA 기준)
5. 여백: 섹션 내부 최소 40px 패딩 (인쇄 후 답답하지 않게)
```

### 색상 규칙
```
1. Primary 컬러: 전체 면적의 30% 이내
2. Secondary 컬러: 포인트 요소에만 사용
3. 본문 텍스트: 항상 #1A1A1A 또는 White (배경에 따라)
4. 인쇄 호환: RGB 기반이나 채도 과도한 네온 컬러 지양
```

## 디자인 참고 원칙

### DO (좋은 전단지)
- 상호명이 1초 안에 눈에 들어옴
- 전화번호가 3초 안에 찾아짐
- 메뉴/가격이 한눈에 정리됨
- 여백이 충분해서 읽기 편함
- 업종 분위기가 즉시 느껴짐

### DON'T (나쁜 전단지)
- 텍스트가 빽빽하게 채워짐
- 폰트가 3종류 이상 혼용
- 색상이 5가지 이상 난무
- 중요 정보가 어디 있는지 모름
- 이미지 품질이 낮거나 깨짐
