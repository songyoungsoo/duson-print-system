# 하단 설명방법 (Bottom Description Method)

## 개요
제품 페이지 하단에 상세 설명 섹션을 통합하고 꾸미는 표준 방법입니다.

## 적용 사례: 봉투 페이지 (envelope/index.php)

### 기본 구조
```html
<!-- 옵셋봉투 및 작업 시 유의사항 통합 섹션 (1200px 폭) -->
<div class="envelope-detail-combined" style="width: 1200px; max-width: 100%; margin: 30px auto; padding: 25px; background: #f8f9fa; border-radius: 12px; border: 1px solid #e0e0e0;">
    <?php include "explane05.php"; ?>
</div>
```

### 핵심 스타일 속성
- **폭 설정**: `width: 1200px` (고정폭)
- **반응형**: `max-width: 100%` (모바일 대응)
- **중앙 정렬**: `margin: 30px auto`
- **내부 여백**: `padding: 25px`
- **배경색**: `background: #f8f9fa` (연한 회색)
- **모서리**: `border-radius: 12px` (둥근 모서리)
- **테두리**: `border: 1px solid #e0e0e0` (연한 테두리)

## 디자인 원칙

### 1. 통합 접근법
- 여러 설명 섹션을 하나의 컨테이너로 합치기
- 기존 PHP include 파일 활용 (예: explane05.php)

### 2. 시각적 분리
- 본문과 구분되는 배경색 사용
- 적절한 여백과 패딩으로 시각적 공간 확보

### 3. 일관성 유지
- 사이트 전체 디자인 시스템과 일치
- Wall Street Professional Style 적용

### 4. 반응형 고려
- 데스크톱: 1200px 고정폭
- 모바일: 100% 폭으로 자동 조정

## 재사용 가능한 CSS 클래스

```css
.product-detail-combined {
    width: 1200px;
    max-width: 100%;
    margin: 30px auto;
    padding: 25px;
    background: #f8f9fa;
    border-radius: 12px;
    border: 1px solid #e0e0e0;
}
```

## 적용 절차

1. **위치 선정**: 제품 페이지 하단, footer 바로 위
2. **컨테이너 생성**: 위 스타일 속성을 가진 div 생성
3. **내용 포함**: 기존 설명 PHP 파일 include
4. **스타일 조정**: 필요에 따라 폭이나 색상 조정

## 주의사항

- 계산 로직 섹션과 분리하여 배치
- 기존 기능에 영향 주지 않도록 주의
- 모바일에서도 가독성 확보

## 확장 가능성

- 다른 제품 페이지에도 동일한 패턴 적용 가능
- 제품별로 고유한 설명 파일 include 가능
- 탭 구조나 아코디언으로 발전 가능
