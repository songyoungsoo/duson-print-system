# 집 컴퓨터에서 Git Pull 하는 방법

## 1. 기본 명령어 (가장 간단)

```bash
cd /var/www/html
git pull origin main
```

## 2. 로컬 변경사항이 충돌할 경우

```bash
# 로컬 변경 버리고 원격 코드로 완전히 덮어쓰기
cd /var/www/html
git fetch origin
git reset --hard origin/main
```

## 3. 데이터베이스 업데이트 (필수!)

집 컴퓨터 DB에도 `flyer_mesu` 컬럼 추가가 필요합니다:

```sql
-- shop_temp, quotation_temp, mlangorder_printauto 테이블에 flyer_mesu 추가
ALTER TABLE shop_temp ADD COLUMN flyer_mesu INT DEFAULT 0 AFTER mesu;
ALTER TABLE quotation_temp ADD COLUMN flyer_mesu INT DEFAULT 0 AFTER mesu;
ALTER TABLE mlangorder_printauto ADD COLUMN flyer_mesu INT DEFAULT 0 AFTER mesu;
```

## 5. 현재 상태 확인

```bash
# 현재 브랜치 및 상태 확인
git status

# 원격 저장소 확인
git remote -v

# 최근 커밋 히스토리 확인
git log --oneline -5
```

## 6. 권한 문제 해결 (필요시)

```bash
# 파일 소유권 변경
sudo chown -R $USER:$USER /var/www/html

# 또는 www-data로 변경 (웹서버용)
sudo chown -R www-data:www-data /var/www/html
```
