# 상품관리 시스템 V1.0 - 사용자 가이드

## 🚀 시스템 접속

### URL
```
http://localhost/admin/product-manager/
```

### 로그인 요구사항
- 관리자 권한 필요
- 세션: `$_SESSION['admin_logged_in'] = true`

## 📊 메인 대시보드 기능

### 1. **상품 목록 테이블**
- **전체 레코드**: 2,932개 (모든 품목 통합)
- **페이지당 표시**: 20개
- **정렬**: 클릭으로 오름차순/내림차순
- **실시간 편집**: 셀 클릭으로 즉시 수정

### 2. **필터링 시스템**
```
품목 선택 → 스타일 선택 → 재질 선택
[전체보기] [명함] [전단지] [봉투] [카다록]...
```

### 3. **검색 기능**
- **통합 검색**: 가격, 수량, 제품코드 동시 검색
- **실시간 검색**: 입력과 동시에 결과 업데이트

### 4. **페이지네이션**
```
< 이전  [1] [2] [3] ... [147]  다음 >
총 2,932개 / 20개씩 / 147페이지
```

## 🎯 주요 사용법

### **상품 정보 수정**
1. 테이블에서 수정할 셀 클릭
2. 새 값 입력
3. Enter 키 또는 다른 셀 클릭으로 저장
4. 자동으로 데이터베이스 반영

### **필터 적용**
1. **품목 선택**: 원하는 품목 드롭다운 선택
2. **스타일 필터**: 선택한 품목의 스타일만 표시
3. **재질 필터**: 선택한 스타일의 재질만 표시
4. **결과 확인**: 필터된 결과 테이블에 표시

### **검색 수행**
1. 검색창에 키워드 입력
2. 가격 숫자 또는 제품코드 입력 가능
3. 실시간으로 매칭 결과 표시

### **가격 필드 설명**
- **기본가격**: `base_price` - 인쇄 기본 비용
- **디자인가격**: `design_price` - 디자인 작업 비용
- **수량**: `quantity` - 제작 수량 (매 단위)
- **인쇄면**: `print_type` - 1(단면), 2(양면)

## 🔧 고급 기능

### **대량 데이터 관리**
- **정렬**: 컬럼 헤더 클릭으로 정렬
- **빠른 네비게이션**: 페이지 번호 직접 클릭
- **필터 조합**: 여러 필터 동시 적용

### **데이터 검증**
- **숫자 필드**: 자동으로 숫자 형식 검증
- **필수 필드**: 빈 값 입력 시 경고
- **실시간 피드백**: 저장 성공/실패 즉시 표시

## ⚠️ 주의사항

### **데이터 수정 시**
1. **백업 권장**: 대량 수정 전 데이터 백업
2. **단계적 수정**: 한 번에 많은 양 수정 지양
3. **검증 확인**: 수정 후 결과 반드시 확인

### **성능 고려사항**
1. **필터 사용**: 전체 데이터보다 필터 적용 권장
2. **검색 키워드**: 구체적인 키워드로 검색
3. **페이지 이동**: 순차적 이동보다 직접 페이지 입력

## 🛠️ 문제 해결

### **자주 발생하는 문제**

#### 1. **데이터가 표시되지 않음**
```
해결책:
1. 브라우저 새로고침
2. 필터 "전체보기" 선택
3. 검색어 삭제 후 재검색
```

#### 2. **수정이 저장되지 않음**
```
해결책:
1. 관리자 로그인 상태 확인
2. 네트워크 연결 확인
3. 브라우저 콘솔에서 오류 확인
```

#### 3. **페이지 로딩이 느림**
```
해결책:
1. 필터 적용으로 데이터 양 줄이기
2. 브라우저 캐시 삭제
3. 서버 성능 확인
```

## 📊 API 엔드포인트 (개발자용)

### **주요 API**
```
GET  /api/get_products.php      - 상품 목록 조회
GET  /api/get_categories.php    - 카테고리 정보
POST /api/update_product.php    - 상품 정보 수정
POST /api/delete_product.php    - 상품 삭제
```

### **파라미터 예시**
```javascript
// 상품 목록 조회
fetch('/api/get_products.php?page=1&limit=20&product=namecard&search=1000')

// 카테고리 조회
fetch('/api/get_categories.php?product=namecard&type=styles')
```
