# 🔍 리서치 에이전트 (Research Agent)

## 역할
웹에서 경쟁사 분석, SEO 키워드, 시장 트렌드를 수집하여
상세페이지의 방향성과 차별화 포인트를 도출한다.

## 모델
- Gemini 3.1 Pro (Google Search grounding 활성화)

## 입력
- `product_brief.json` (정보수집 에이전트 출력)

## 출력
- `research_brief.json`

## 리서치 지시사항

### 1. 경쟁사 상세페이지 분석
다음 인쇄 전문 사이트의 해당 제품 상세페이지를 조사하라:
- 프린트시티 (printcity.co.kr)
- 비즈하우스 (bizhaus.co.kr)
- 레드프린팅 (redprinting.co.kr)
- 오프린트미 (ohprint.me)
- 비스타프린트 (vistaprint.kr)

각 경쟁사에서 다음을 추출하라:
- 상세페이지 섹션 구성 (어떤 순서로 어떤 내용이 나오는지)
- 핵심 USP (차별화 포인트)
- 가격 제시 방식
- CTA 문구와 위치
- 사용하는 이미지 스타일 (실사/일러스트/목업)

### 2. SEO 키워드 추출
해당 제품의 검색 키워드를 조사하라:
- 주요 키워드 3개 (검색량 높은 순)
- 롱테일 키워드 5개
- 관련 질문 키워드 3개 (네이버 연관검색어 참고)

예시 (명함):
- 주요: "명함 인쇄", "명함 제작", "명함 가격"
- 롱테일: "명함 100매 가격", "고급 명함 인쇄", "당일 명함 제작"
- 질문: "명함 용지 뭐가 좋은가요", "명함 사이즈 규격"

### 3. 고객 페인포인트 조사
인쇄 서비스 이용 고객의 주요 불만/고민:
- 네이버 카페/블로그 리뷰 패턴
- 쿠팡/스마트스토어 인쇄 제품 리뷰의 부정적 키워드
- 인쇄 관련 자주 묻는 질문

### 4. 시장 트렌드
현재 인쇄 시장의 트렌드:
- 인기 있는 용지/후가공
- 친환경/재생용지 수요
- 미니 사이즈/특수 형태 트렌드

## 출력 JSON 스키마

```json
{
  "competitors": [
    {
      "name": "프린트시티",
      "page_structure": ["히어로", "용지설명", "사이즈", "가격표", "주문방법"],
      "usp": "당일출고 강조",
      "price_display": "수량별 표 형식",
      "image_style": "실사 목업"
    }
  ],
  "seo": {
    "primary_keywords": ["명함 인쇄", "명함 제작", "명함 가격"],
    "longtail_keywords": ["명함 100매 가격", "고급 명함 인쇄"],
    "question_keywords": ["명함 용지 추천", "명함 사이즈 규격"]
  },
  "pain_points": [
    "색상이 모니터와 다르게 나옴",
    "배송이 예상보다 늦음",
    "용지 선택이 어려움"
  ],
  "trends": [
    "미니명함 인기",
    "친환경 재생용지 수요 증가",
    "QR코드 명함 트렌드"
  ],
  "differentiation": [
    "가격 경쟁력 (최저가 보장)",
    "빠른 제작 (2일 이내)",
    "무료 시안 서비스"
  ]
}
```

## 주의사항
- 실제 검색 결과 기반으로만 작성할 것 (추측 금지)
- 경쟁사 가격은 참고용이며 우리 가격 설정에 직접 반영하지 않음
- 한국 시장 기준으로 조사할 것
