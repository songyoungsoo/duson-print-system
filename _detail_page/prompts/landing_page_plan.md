# 13섹션 상세페이지 구조 — 인쇄 제품 특화

> 이 문서는 카피라이팅/디자인 에이전트의 시스템 프롬프트에 포함됩니다.
> 각 섹션의 목적, 구성 요소, 심리적 트리거를 정의합니다.

## 전체 흐름 (고객 심리 퍼널)

```
주의 끌기 → 공감 형성 → 문제 인식 → 해결책 제시 → 제품 설명 →
상세 스펙 → 활용 사례 → 가격 납득 → 프로세스 이해 → 사회적 증거 →
의문 해소 → 신뢰 확보 → 행동 유도
```

---

## Section 1: 긴급성 헤더 (Urgency Header)
- **목적**: 즉각적 주의 끌기 + 이탈 방지
- **심리 트리거**: 희소성, FOMO (Fear of Missing Out)
- **필수 요소**:
  - 기간 한정 혜택 또는 수량 한정 문구
  - 시각적 카운트다운 느낌
  - 강렬한 배경색 (브랜드 Primary 또는 Red)
- **텍스트 예시**: "이번 주 주문 시 10% 할인 | 3월 7일까지"
- **이미지 구성**: 풀 배경 + 중앙 대형 텍스트 + 작은 서브텍스트

## Section 2: 공감 섹션 (Empathy)
- **목적**: 고객이 "나 얘기다" 느끼게 하기
- **심리 트리거**: 미러링, 감정적 연결
- **필수 요소**:
  - 타겟 고객의 구체적 상황 묘사
  - 질문형 헤드라인 (공감 유도)
  - 사람 이미지 (고민하는 모습)
- **텍스트 예시**: "명함이 필요한데 어디서 맞춰야 할지 모르겠다면?"
- **이미지 구성**: 사무실/카페 배경 + 고민하는 사람 + 텍스트 오버레이

## Section 3: 문제 정의 (Problem Definition)
- **목적**: 기존 경험의 불편함을 구체화
- **심리 트리거**: 고통 회피 동기
- **필수 요소**:
  - 문제점 3가지 (아이콘 + 짧은 설명)
  - X 마크 또는 부정적 시각 요소
  - 어두운/차가운 톤
- **인쇄 업계 공통 문제**:
  1. 비싼 가격 — "소량 주문하면 단가가 너무 높다"
  2. 느린 배송 — "급하게 필요한데 5~7일이나 걸린다"
  3. 품질 불만 — "색상이 화면과 달라서 다시 맞춰야 했다"
- **이미지 구성**: 3분할 레이아웃 + X 아이콘 + 문제 텍스트

## Section 4: 솔루션 제시 (Solution)
- **목적**: "우리가 해결합니다" — 희망 전환점
- **심리 트리거**: 안도감, 문제 해결 기대
- **필수 요소**:
  - 체크마크/O 마크
  - 밝은 톤으로 전환
  - 두손기획인쇄의 USP 3가지
- **인쇄 업계 USP**:
  1. ✓ 합리적 가격 — "100매 3,000원부터"
  2. ✓ 빠른 제작 — "주문 후 2일 이내 출고"
  3. ✓ 정확한 색상 — "최신 인쇄기로 색상 100% 재현"
- **이미지 구성**: 밝은 배경 + 3열 체크 카드 + 브랜드 컬러 강조

## Section 5: 제품 소개 (Product Features)
- **목적**: 핵심 특징을 임팩트 있게 전달
- **심리 트리거**: 가치 인식
- **필수 요소**:
  - 제품 실물 목업 이미지 (중앙)
  - 특징 3개가 제품을 둘러싸는 레이아웃
  - 클린 화이트 배경
- **이미지 구성**: 중앙 제품 목업 + 주변 3개 특징 포인트

## Section 6: 스펙 상세 (Specifications)
- **목적**: 구체적 정보 제공 → 구매 확신
- **심리 트리거**: 정보 충족, 전문성 인식
- **필수 요소**:
  - 용지 종류별 카드 (이미지+설명)
  - 사이즈 비교 시각화
  - 후가공 옵션 (박, 형압, 에폭시 등)
- **이미지 구성**: 카드 그리드 (2~3열) + 실물 질감 클로즈업

## Section 7: 비포&애프터 / 활용 사례 (Use Cases)
- **목적**: 실제 사용 장면 → 구매 욕구
- **심리 트리거**: 시각적 증거, 상상력 자극
- **필수 요소**:
  - Before/After 또는 활용 장면 3가지
  - 실제 환경에서의 제품 사진
  - 짧은 캡션 ("카페 카운터에 비치된 명함")
- **이미지 구성**: 3분할 라이프스타일 이미지 + 캡션

## Section 8: 가격 안내 (Pricing)
- **목적**: 가격 투명성 → 가격 저항 해소
- **심리 트리거**: 가치 대비 가격 납득
- **필수 요소**:
  - 수량별 가격 표 (3~5개 구간)
  - "가장 인기" 배지 (추천 수량 강조)
  - 개당 단가 환산 ("개당 30원")
- **이미지 구성**: 가격 표 + 추천 강조 + CTA 미니 버튼

## Section 9: 제작 과정 (Process)
- **목적**: 불안 해소 — "이렇게 진행됩니다"
- **심리 트리거**: 예측 가능성, 통제감
- **필수 요소**:
  - 4단계: ① 주문 → ② 시안확인 → ③ 인쇄 → ④ 배송
  - 각 단계별 아이콘 + 소요시간
  - 화살표/타임라인 연결
- **이미지 구성**: 가로 4단계 인포그래픽 + 아이콘

## Section 10: 고객 후기 (Reviews)
- **목적**: 사회적 증거 — "다른 사람도 만족했다"
- **심리 트리거**: 밴드왜건 효과, 안심
- **필수 요소**:
  - 후기 카드 3개 (별점 + 텍스트 + 프로필)
  - 구체적 직업/용도 언급
  - 총 평점 요약 ("평균 4.8/5.0")
- **이미지 구성**: 3열 후기 카드 + 별점 + 프로필 아이콘

## Section 11: FAQ (자주 묻는 질문)
- **목적**: 마지막 의문 해소 → 이탈 방지
- **심리 트리거**: 정보 완결성
- **필수 요소**:
  - Q&A 5개
  - 아코디언 UI 스타일
  - 간결한 답변 (2줄 이내)
- **공통 FAQ**:
  1. 주문 후 얼마나 걸리나요?
  2. 시안 수정은 몇 번까지 가능한가요?
  3. 소량 주문도 가능한가요?
  4. 색상이 화면과 다르면 어떻게 하나요?
  5. 결제 방법은 무엇이 있나요?
- **이미지 구성**: Q&A 리스트 레이아웃 + 회색 배경

## Section 12: 신뢰 배지 (Trust Badges)
- **목적**: 최종 안심 요소
- **심리 트리거**: 안전감, 리스크 제거
- **필수 요소**:
  - 4개 배지: 안전결제 / 품질보증 / 전국배송 / 교환환불
  - 각 배지에 아이콘 + 한 줄 설명
  - 깔끔한 가로 배치
- **이미지 구성**: 4열 아이콘 배지 + 설명 텍스트

## Section 13: 최종 CTA (Call to Action)
- **목적**: 구매 행동 유도 — 마지막 push
- **심리 트리거**: 행동 동기, 마감 효과
- **필수 요소**:
  - 크고 눈에 띄는 CTA 버튼
  - 가격 한 번 더 상기
  - 추가 혜택 리마인더 ("빠른 제작 + 안전 택배")
  - 강렬한 Accent 컬러 배경
- **텍스트 예시**: "지금 바로 주문하기 → 100매 3,000원부터"
- **이미지 구성**: Accent 배경 + 대형 CTA 버튼 + 가격 + 화살표
