# Presentation Design Skill (Frontend-Driven)

This skill adapts modern frontend web design principles (UI/UX) to create visually striking, production-grade presentation slides. It avoids generic "AI slop" aesthetics (like plain bulleted lists on white backgrounds).

## 1. Typography (HARD RULE: PRETENDARD ONLY)
- **Primary & Secondary Font:** MUST USE **Pretendard** exclusively for all text.
- **Font Weights for Contrast:** Since the font family is fixed to Pretendard, you MUST create visual hierarchy using extreme weight and size contrast.
  - Titles (H1): `Pretendard ExtraBold` or `Pretendard Bold` (36pt - 60pt).
  - Section Headers (H2): `Pretendard SemiBold` (20pt - 24pt).
  - Body Text (P): `Pretendard Regular` or `Pretendard Light` (14pt - 16pt).
  - *Never use Arial, Calibri, Malgun Gothic, or Inter.*

## 2. Spatial Composition & Layout (Frontend Grids)
Apply web CSS Grid/Flexbox thinking to slide layouts:
- **Unexpected Layouts:** Break the standard "Title at top, content below" mold. Use split screens (50/50 ratio), asymmetrical columns (30/70), or full-bleed imagery with overlay text.
- **Whitespace (Negative Space):** Use generous padding and margins (equivalent to `padding: 2rem` or `margin: 3rem`). Do not crowd the slide. Let elements breathe.
- **Alignment:** Strictly align elements to an invisible grid. Left-align body text (never center body paragraphs unless it's a massive, single-line pull quote).

## 3. Color & Theme
- **Dominance over Equality:** Pick a BOLD aesthetic direction. One color should dominate (60-70%), one supporting tone (20-30%), and one sharp accent color (5-10%).
- **Contrast:** Use high contrast. Dark mode slides (e.g., Charcoal background with White text and Neon accent) often look more premium and modern than white slides.
- **Gradients & Textures:** Use subtle gradient backgrounds or mesh gradients instead of flat, dead colors if the tone permits.

## 4. Visual Elements & UI Components
Treat slide elements like modern UI components:
- **Cards:** Group related information into "cards" (shapes with subtle background colors, rounded corners, and soft drop shadows).
- **Pills/Badges:** Use small rounded rectangles with text inside for tags, categories, or status indicators.
- **Icons:** Use modern, minimalist vector icons (like React-icons or Phosphor style) placed inside circular containers to create visual anchor points.
- **Data Viz / Big Numbers:** Render statistics as "Big Number Callouts" (huge font size for the number, small muted text for the label) rather than boring tables.

## 5. What to AVOID (The "AI Slop")
- Text-heavy slides that look like Word documents pasted into a slide.
- Default PowerPoint templates, themes, or 3D generic cliparts.
- Generic accent lines under titles.
- Multiple primary colors fighting for attention.
