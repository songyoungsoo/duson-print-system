# 13섹션 상세페이지 구조 — 인쇄 제품 특화 (V2)

> V2 규격: 1100×800px 캔버스, 내용 영역 800×800px (좌우 150px 여백)
> 이 문서는 카피라이팅/디자인 에이전트의 시스템 프롬프트에 포함됩니다.

## 전체 흐름 (고객 심리 퍼널)

```
주의 끌기 → 공감 형성 → 문제 인식 → 해결책 제시 → 제품 설명 →
상세 스펙 → 활용 사례 → 가격 납득 → 프로세스 이해 → 사회적 증거 →
의문 해소 → 신뢰 확보 → 행동 유도
```

---

## Section 1: 긴급성 헤더 (Urgency Header)
- **목적**: 즉각적 주의 끌기 + 이탈 방지
- **심리 트리거**: 희소성, FOMO
- **필수 요소**: 기간 한정 혜택, 카운트다운 느낌, 강렬한 배경색
- **이미지 구성**: 풀 배경 + 중앙 대형 텍스트 + 작은 서브텍스트

## Section 2: 공감 섹션 (Empathy)
- **목적**: 고객이 "나 얘기다" 느끼게 하기
- **심리 트리거**: 미러링, 감정적 연결
- **필수 요소**: 타겟 고객의 구체적 상황 묘사, 질문형 헤드라인
- **이미지 구성**: 배경 + 고민하는 사람 + 텍스트 오버레이

## Section 3: 문제 정의 (Problem Definition)
- **목적**: 기존 경험의 불편함을 구체화
- **심리 트리거**: 고통 회피 동기
- **문제점 3가지**: 비싼 가격, 느린 배송, 품질 불만
- **이미지 구성**: 3분할 레이아웃 + X 아이콘 + 문제 텍스트

## Section 4: 솔루션 제시 (Solution)
- **목적**: "우리가 해결합니다" — 희망 전환점
- **심리 트리거**: 안도감, 문제 해결 기대
- **USP 3가지**: 합리적 가격, 빠른 제작, 정확한 색상
- **이미지 구성**: 밝은 배경 + 3열 체크 카드 + 브랜드 컬러

## Section 5: 제품 소개 (Product Features)
- **목적**: 핵심 특징을 임팩트 있게 전달
- **심리 트리거**: 가치 인식
- **이미지 구성**: 중앙 제품 목업 + 주변 3개 특징 포인트

## Section 6: 스펙 상세 (Specifications)
- **목적**: 구체적 정보 제공 → 구매 확신
- **심리 트리거**: 정보 충족, 전문성 인식
- **이미지 구성**: 카드 그리드 (2~3열) + 실물 질감 클로즈업

## Section 7: 비포&애프터 / 활용 사례 (Use Cases)
- **목적**: 실제 사용 장면 → 구매 욕구
- **심리 트리거**: 시각적 증거, 상상력 자극
- **이미지 구성**: 3분할 라이프스타일 이미지 + 캡션

## Section 8: 가격 안내 (Pricing)
- **목적**: 가격 투명성 → 가격 저항 해소
- **심리 트리거**: 가치 대비 가격 납득
- **이미지 구성**: 가격 표 + 추천 강조 + CTA 미니 버튼

## Section 9: 제작 과정 (Process)
- **목적**: 불안 해소 — "이렇게 진행됩니다"
- **심리 트리거**: 예측 가능성, 통제감
- **4단계**: 주문 → 시안확인 → 인쇄 → 배송
- **이미지 구성**: 가로 4단계 인포그래픽 + 아이콘

## Section 10: 고객 후기 (Reviews)
- **목적**: 사회적 증거
- **심리 트리거**: 밴드왜건 효과, 안심
- **이미지 구성**: 3열 후기 카드 + 별점 + 프로필 아이콘

## Section 11: FAQ (자주 묻는 질문)
- **목적**: 마지막 의문 해소 → 이탈 방지
- **심리 트리거**: 정보 완결성
- **이미지 구성**: Q&A 리스트 레이아웃

## Section 12: 신뢰 배지 (Trust Badges)
- **목적**: 최종 안심 요소
- **심리 트리거**: 안전감, 리스크 제거
- **4개 배지**: 안전결제 / 품질보증 / 전국배송 / 교환환불
- **이미지 구성**: 4열 아이콘 배지 + 설명 텍스트

## Section 13: 최종 CTA (Call to Action)
- **목적**: 구매 행동 유도 — 마지막 push
- **심리 트리거**: 행동 동기, 마감 효과
- **이미지 구성**: Accent 배경 + 대형 CTA 버튼 + 가격 + 화살표
