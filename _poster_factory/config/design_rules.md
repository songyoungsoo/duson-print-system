## 1. 시선 흐름 패턴 (Eye Flow)

### Z-패턴 (이미지 중심 전단지)
```
좌상 (상호/로고)  ──────→  우상 (배지/태그)
                    ↘
좌하 (특징/서비스)  ──────→  우하 (CTA/연락처) ← 최종 행동 유도!
```
- **적용 레이아웃**: classic_grid, hero_dominant, bold_typo
- **핵심 규칙**: CTA/연락처는 반드시 하단 우측(Z의 끝점)에 배치
- **히어로**: 좌상 시작점에 상호명+핵심 카피

### F-패턴 (텍스트/리스트 중심 전단지)
```
━━━━━━━━━━━━━━━━━  (상단 수평 스캔 — 헤드라인)
┃
┃━━━━━━━━━━━━━━━   (중단 짧은 수평 — 서비스A)
┃
┃━━━━━━━━━━       (하단 짧은 수평 — 서비스B)
```
- **적용 레이아웃**: magazine_split, side_by_side
- **핵심 규칙**: 제목/핵심문장을 좌측에, 중요 키워드를 상단에 배치
- **불릿 리스트**: 좌측 정렬, 스캔 가능한 짧은 문구

### 구텐베르크 4분면
| 좌상 (1차 주목) | 우상 (2차 주목) |
|:---:|:---:|
| **상호명/로고** | **이벤트 배지** |
| 좌하 (낮은 주목) | 우하 (터미널) |
| 부가 정보 | **CTA/전화번호** |

## 2. 타이포그래피 위계

### 황금비율 기반 폰트 크기 (2130×3000px 캔버스)

본문 크기에 황금비율(×1.618)을 적용하여 위계를 형성:

| 요소 | 크기(px) | 굵기 | 근거 | 용도 |
|------|---------|------|------|------|
| 상호명 (볼드 타이포) | 120~160 | 700 | 최대 강조 | bold_typo 레이아웃 전용 |
| 히어로 헤드라인 | 72 | 700 | — | 히어로 내 메인 카피 |
| CTA 헤드라인 | 48~56 | 700 | — | "지금 방문하세요" |
| 슬로건 | 48~56 | 700 | 본문×1.618² | 독립 슬로건 섹션 |
| 가격 (강조) | 48~56 | 700 | — | 가격표, 이벤트 가격 |
| 전화번호 | 42~48 | 700 | — | 연락처 (자간 2-3px 추가) |
| 품목명 | 36~44 | 700 | 본문×1.618 | 품목 이름 |
| 특징/불릿 | 28~32 | 700 | — | 특징 스트립, 카테고리 뱃지 |
| 품목 설명 | 26~30 | 400 | 기준 본문 | 일반 설명 텍스트 |
| 서브 설명 | 22~26 | 300 | — | 부가 정보 |
| 주소/시간 | 28~32 | 400 | — | CTA 내 부가 연락처 |

### 행간 (line-height)
- **제목**: 폰트 크기 × 1.1~1.2 (타이트하게)
- **본문**: 폰트 크기 × 1.4~1.5 (여유있게)
- **불릿 리스트**: 항목 간 55px 간격

### 자간 (letter-spacing)
- **슬로건, 배너**: 2~5px (시원하고 넓은 느낌)
- **상호명**: 0~3px
- **본문**: 0~1px
- **가격 숫자**: 0px (숫자는 벌리지 않음)
- **전화번호**: 2~3px (가독성 향상)
- **한글 일반**: -2% ~ 0% (견고한 느낌)

### 폰트 굵기 3단계 분리
```
700 (Bold)    — 상호명, 헤드라인, 가격, 전화번호, CTA
400 (Regular) — 품목 설명, 주소, 영업시간
300 (Light)   — 서브 설명, 부가 정보
```

### 인쇄물 폰트 최소 크기
- 제목: 14pt(≈47px) 이상 권장
- 본문: 9pt(≈30px) 이상 유지
- 300weight 이하 폰트는 인쇄 시 번질 수 있음 → 400weight 이상 사용

## 3. 색상 사용 규칙

### 3.1 배경
- 기본: 밝은 오프화이트 (#FAFAF8 ~ #F5F5F5)
- 섹션 구분: 브랜드 컬러 4~8% opacity 배경
- CTA 영역: 브랜드 컬러 솔리드 배경 + 흰색 텍스트
- 2026 트렌드: 저채도 자연색 배경 (Soft Brown, Misty Blue)

### 3.2 강조
- 액센트 컬러: 가격, CTA 헤드라인, 프로모 텍스트에만 사용
- 프로모 배너: 액센트 컬러 솔리드 배경 + 흰색 Bold 텍스트
- 특징 스트립: 브랜드 컬러 8% opacity 배경 + 진한 텍스트
- **액센트 면적**: 전체의 10~15% 이내 (과도하면 주목성 상실)

### 3.3 대비
- **최소 대비 비율**: 4.5:1 (일반 텍스트), 3:1 (큰 텍스트 48px+)
- 본문 텍스트: 진한 차콜 (#2D3436 ~ #333333)
- opacity로 위계 표현: 1.0 (제목) → 0.65 (설명) → 0.5 (서브)
- 흰색 배경 위 검정 텍스트: 21:1 대비 (최적)

### 3.4 색상 심리 활용 (업종별)
| 색상 | 심리 효과 | 적합 업종 |
|------|----------|----------|
| 빨강/주황 | 식욕 자극, 흥분, 긴급 | 음식점, 분식, 배달 |
| 파랑/남색 | 신뢰, 안정, 전문성 | 학원, 교육, 병원 |
| 베이지/갈색 | 따뜻함, 편안, 고급 | 카페, 베이커리 |
| 핑크/퍼플 | 여성적, 세련, 창의 | 뷰티, 네일, 에스테틱 |
| 블랙/다크 | 강렬, 파워, 역동 | 헬스장, PT |
| 그린/틸 | 건강, 청결, 자연 | 병원, 한의원, 웰니스 |
| 노랑/골드 | 주목, 낙관, 에너지 | 이벤트, 프로모 포인트 |

### 3.5 CMYK 인쇄 주의사항
- RGB → CMYK 변환 시 5~10% 어두워짐
- 리치블랙(인쇄용 진한 검정): C=60, M=60, Y=60, K=100
- 인쇄 해상도: 300dpi 이상
- 현재 포스터 공장은 RGB 출력 (고객이 Illustrator에서 CMYK 변환)

## 4. 레이아웃 패턴별 특성

### classic_grid (클래식 그리드)
- 가장 안전한 기본 레이아웃 — **미리캔버스/캔바 음식점/카페 기본 구조와 일치**
- 히어로 49% + 품목 그리드 25% + CTA 12%
- 품목 3~6개에 최적
- 슬로건 밴드로 히어로↔품목 전환 부드럽게
- **Z패턴**: 히어로 → 슬로건 → 품목 → CTA

### hero_dominant (히어로 도미넌트)
- 히어로가 62% — **이미지 퀄리티가 전단지 성패를 결정하는 업종**
- 품목은 아이콘+텍스트 가로 리스트로 축약
- Before/After 구조도 가능
- **Z패턴 변형**: 대형 히어로 → 슬로건 → 품목 수평 → CTA

### magazine_split (매거진 분할)
- 정보량 많은 업종 (학원: 과목 10개+) — **크몽: '신뢰감 핵심'**
- 색띠로 섹션 구분 → 시각적 정리
- 불릿 리스트가 핵심 요소
- 히어로 35% (상대적으로 작음)
- **F패턴**: 상단 헤드라인 → 섹션A 리스트 → 섹션B 리스트 → CTA

### bold_typo (볼드 타이포)
- 상호명이 화면의 25%를 차지 — **글자 자체가 비주얼**
- 가격표가 핵심 정보
- 음식 누끼 이미지와 큰 가격의 조합
- 프로모 배너도 크게 (10%)
- **Z패턴 강화**: 초대형 상호 → 음식사진 → 가격표 → CTA

### side_by_side (좌우 분할)
- 좌: 대형 인물/시설 이미지 (50%) — **1인 전문가, 미용실에 최적**
- 우: 서비스 가격 리스트
- 하단 30%는 풀폭 프로모+CTA
- **F패턴 변형**: 좌측 이미지 → 우측 상호명 → 우측 서비스 리스트 → CTA

## 6. CTA (Call-to-Action) 배치 원칙

### 위치
- Z패턴 종점(하단 우측)에 배치 — **반응률 최고**
- 구텐베르크 다이어그램 '터미널 영역' 활용
- CTA 주변 최소 여백 40px

### 구성요소 (위에서 아래로)
1. **CTA 헤드라인** — 행동 유도 문구 ("지금 방문하세요", "전화주세요")
2. **전화번호** — 가장 크게 (42~48px Bold, 자간 2-3px)
3. **주소** — 간결하게 (역/건물명 중심)
4. **영업시간** — 요일별 표기
5. **SNS** — 인스타/블로그 (선택)

### 시각적 강조
- CTA 영역: 브랜드 컬러 솔리드 배경 + 흰색 텍스트
- 상단 분리선: 액센트 컬러 2px, 중앙 300px 폭
- 전화번호 아이콘 (선택)

## 7. 2026 디자인 트렌드 반영

### 핵심 키워드
- **감성 회복** — 저채도, 자연색, 시각적 피로 감소
- **신 미니멀리즘** — 넓은 여백 + 1~2가지 강한 색상
- **오가닉 플로우** — 자연스러운 곡선, 유동적 레이아웃 (카페/뷰티)
- **듀얼 미학** — 클래식+모던 대비 (전통+현대)
- **대담한 타이포** — 헤드라인은 크고 대담, 본문은 편안

### 적용 방법
1. 기본 배경을 순백(#FFFFFF) 대신 오프화이트(#FAFAF8~#F5F0EB) 사용
2. 액센트 컬러는 저채도 자연색 옵션 추가 (세이지 그린, 미스티 블루)
3. 여백을 넉넉하게 — overcrowding 금지 (Canva 가이드 핵심 원칙)
4. 하나의 핵심 메시지만 전달 — 전단지 1장에 1개 핵심 메시지
5. CTA를 명확하게 — 고객에게 원하는 행동을 직접 지시

## 8. 장식 요소

### 구분선
- 슬로건 위아래: accent 컬러 2px, opacity 0.4
- 섹션 경계: brand 컬러 1px, opacity 0.15
- CTA 상단: accent 컬러 2px, 중앙 300px 폭

### 배경 패턴
- 섹션별 brand 컬러 4~8% opacity 사각형
- 프로모 배너: accent 컬러 솔리드 + rx="12" 라운드

### 재단선 (Trim marks)
- 블리드 15px 위치에 0.5px 회색 라인
- 4 모서리 × 2 (가로+세로) = 8개 마크

## 9. 이미지 규칙

### 히어로
- aspect_ratio: "5:4" (가로형, 포스터 상단 맞춤)
- 상호명 + 슬로건 텍스트를 Gemini가 직접 렌더링
- preserveAspectRatio="xMidYMid meet" (slice 금지!)

### 품목 이미지
- aspect_ratio: "1:1" (정사각형)
- 텍스트 없이 순수 제품/서비스 사진
- SVG `<text>`로 품목명/설명 별도 추가

### 이미지 임베드
- `--embed` 플래그: base64 data URI로 SVG에 임베드
- 임베드 안 하면: `images/hero.png` 상대 경로 참조
- Illustrator에서 열려면 임베드 필수 (WSL 경로 깨짐)

### 이미지 품질
- 최소 해상도: 300dpi 인쇄 기준
- Gemini 생성 이미지: 포토리얼리스틱 스타일 필수
- 음식 사진: 자연광, 따뜻한 톤, 높은 채도
- 인물 사진: 전문적 조명, 배경 정리

## 10. 그리드 시스템 (2130×3000px 캔버스)

|------|------|------|

### 열(Column) 시스템

## 11. 레이아웃 선택 의사결정 트리

```
입력: 업종 + 품목 수 + 사용자 선호

1. 사용자가 레이아웃을 지정했는가?
   → YES: 지정된 레이아웃 사용
   → NO: 2번으로

2. 업종 매칭 (layout_patterns.json → by_industry)
   → 매칭되는 업종의 첫 번째 추천 레이아웃 선택
   → 매칭 안 되면: 3번으로

3. 품목 수 기반 (by_item_count)
   → 1-2개: hero_dominant (이미지 강조)
   → 3-4개: classic_grid (균형)
   → 5-6개: classic_grid 또는 magazine_split
   → 7+개: magazine_split (정보 정리)

4. 최종 fallback: classic_grid
```
