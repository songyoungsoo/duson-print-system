# 인쇄원가계산 3 - 인쇄비

## 📐 기본 개념

### 합판 vs 독판

| 구분 | 합판인쇄 | 독판인쇄 |
|------|---------|---------|
| 대상 | 90g 전용 | 120g 이상 |
| 방식 | 여러 건 모아서 공동 인쇄 | 단독 인쇄 |
| 장점 | 인쇄비 저렴 | 품질/일정 자유도 |
| 최소 판 | - | A2 |
| 판비 | A2 등분 계산 | 규격별 별도 |

### 8색 인쇄기 구조
```
┌─────────────────────────────────────────┐
│  8색 인쇄기                              │
├─────────────────────────────────────────┤
│  [C][M][Y][K] ← 앞면 4도                │
│  [C][M][Y][K] ← 뒷면 4도                │
├─────────────────────────────────────────┤
│  단면 4도: 4색만 사용 → 비효율           │
│  양면 4도+4도: 8색 풀가동 → 효율적       │
└─────────────────────────────────────────┘

→ 양면 인쇄가 2배가 아닌 1.5배인 이유!
```

## 📋 합판인쇄 가격표 (90g 전용)

> 여러 건을 모아서 공동 인쇄 → 인쇄비 저렴

### 판비 (A2 32,000원 기준 등분)

| 규격 | 등분 | 4도 단면 | 4도 양면 (×2) |
|------|------|----------|--------------|
| A2 | 1 | 32,000원 | 64,000원 |
| A3 | 2 | 16,000원 | 32,000원 |
| A4 | 4 | 8,000원 | 16,000원 |
| A5~A6 | 4 | 8,000원 | 16,000원 |

※ 최소 4등분 (A4) 적용

### 인쇄비 (1연당, A/B시리즈 동일)

| 인쇄 방식 | 가격 | 배율 |
|----------|------|------|
| 단면 4도 | 10,000원 | ×1.0 |
| 양면 4도+4도 | 15,000원 | ×1.5 |

## 📋 독판인쇄 가격표 (120g 이상)

> 단독 인쇄, 최소 판 크기 = A2

### 판비 (규격별)

| 규격 | 4도 단면 | 4도 양면 (×2) |
|------|----------|--------------|
| A2 | 32,000원 | 64,000원 |
| A3 | 22,000원 | 44,000원 |
| A4 | 12,000원 | 24,000원 |

### 인쇄비 (1연당, A/B시리즈 동일)

| 인쇄 방식 | 가격 | 배율 |
|----------|------|------|
| 단면 4도 | 40,000원 | ×1.0 |
| 양면 4도+4도 | 60,000원 | ×1.5 |

## 📊 합판 vs 독판 비교 (A4 기준)

| 항목 | 합판 (90g) | 독판 (120g+) | 배율 |
|------|-----------|-------------|------|
| 판비 단면 | 8,000원 | 12,000원 | ×1.5 |
| 판비 양면 | 16,000원 | 24,000원 | ×1.5 |
| 인쇄비 단면 | 10,000원 | 40,000원 | ×4 |
| 인쇄비 양면 | 15,000원 | 60,000원 | ×4 |

## 🧮 인쇄비 계산 공식

### 합판 (90g)
```
판비 = A2 32,000원 ÷ 등분 × 면수(1 or 2)
인쇄비 = 연수 × 10,000원 (단면)
인쇄비 = 연수 × 15,000원 (양면)
```

### 독판 (120g+)
```
판비 = 규격별 가격 × 면수(1 or 2)
인쇄비 = 연수 × 40,000원 (단면)
인쇄비 = 연수 × 60,000원 (양면)
```

## 📝 계산 예시

### 예시 1: 90g A4 전단지 4,000장 단면 4도
```
A4 = 8절 → 1연당 4,000장
4,000장 ÷ 4,000장 = 1연

종이값 = 1연 × 35,000원 = 35,000원
판비 = 32,000 ÷ 4 = 8,000원
인쇄비 = 1연 × 10,000원 = 10,000원

총원가 = 35,000 + 8,000 + 10,000 = 53,000원
```

### 예시 2: 90g A4 전단지 4,000장 양면 4도
```
A4 = 8절 → 1연당 4,000장
4,000장 ÷ 4,000장 = 1연

종이값 = 1연 × 35,000원 = 35,000원
판비 = 32,000 ÷ 4 × 2 = 16,000원
인쇄비 = 1연 × 15,000원 = 15,000원

총원가 = 35,000 + 16,000 + 15,000 = 66,000원
```

### 예시 3: 150g A4 전단지 4,000장 양면 4도
```
A4 = 8절 → 1연당 4,000장
4,000장 ÷ 4,000장 = 1연

종이값 = 1연 × 58,333원 = 58,333원
판비 = 12,000 × 2 = 24,000원 (독판)
인쇄비 = 1연 × 60,000원 = 60,000원

총원가 = 58,333 + 24,000 + 60,000 = 142,333원
```

### 예시 4: 비용 비교 (A4 4,000장 양면 4도)

| 항목 | 90g 합판 | 150g 독판 |
|------|---------|----------|
| 종이값 | 35,000원 | 58,333원 |
| 판비 | 16,000원 | 24,000원 |
| 인쇄비 | 15,000원 | 60,000원 |
| **총원가** | **66,000원** | **142,333원** |

## ⚠️ 중요 포인트

### A/B시리즈 인쇄비 동일
```
종이값: B = A × 1.5 (면적 비례)
판비:   합판 동일, 독판 B = A × 1.5
인쇄비: B = A ← 관행상 동일 (억울하지만...)
```

### 양면 = 1.5배 (2배 아님!)
- 8색 인쇄기 효율성 때문
- 단면: 4색만 사용 (비효율)
- 양면: 8색 풀가동 (효율적)
