# 인쇄원가계산 4 - 후가공비

## 📐 기본 개념

### 후가공 종류
- **코팅**: 표면 보호 및 광택 처리
- **접지**: 종이 접기
- **재단**: 종이 자르기

### 가격 기준
- 모든 후가공비는 **1연당** 가격
- A/B시리즈 동일 가격 (관행상)

---

## 📋 코팅비 (1연당)

### 가격표

| 종류 | 단면 | 양면 (×2) |
|------|------|----------|
| 유광코팅 | 80,000원 | 160,000원 |
| 무광코팅 | 90,000원 | 180,000원 |

### 유광 vs 무광

| 구분 | 유광코팅 | 무광코팅 |
|------|---------|---------|
| 외관 | 반짝반짝 광택 | 차분한 매트 |
| 가격 | 80,000원 | 90,000원 |
| 용도 | 사진, 화보 | 고급 인쇄물 |

### ⚠️ 코팅 제한사항

```
┌─────────────────────────────────────────┐
│  150g 이상만 코팅 가능!                  │
│                                         │
│  90g, 120g → ❌ 코팅 불가 (종이가 말림)  │
│  150g 이상 → ✅ 코팅 가능                │
└─────────────────────────────────────────┘
```

| 평량 | 코팅 가능 여부 |
|------|--------------|
| 90g | ❌ 불가 |
| 120g | ❌ 불가 |
| 150g | ✅ 가능 |
| 180g | ✅ 가능 |
| 200g | ✅ 가능 |
| 250g | ✅ 가능 |
| 300g | ✅ 가능 |

## 📋 접지비 (1연당)

### 가격표

| 접지 단수 | 가격 | 비고 |
|----------|------|------|
| 2단 | 40,000원 | |
| 3단 | 40,000원 | |
| 4단 | 60,000원 | +20,000 |
| 5단 | 80,000원 | +20,000 |
| 6단 | 100,000원 | +20,000 |
| 7단 | 120,000원 | +20,000 |
| n단 | +20,000원씩 | |

### 접지 단수 계산

```
3단 이하: 40,000원 (기본)
4단 이상: 40,000 + (단수-3) × 20,000원

예) 5단 접지 = 40,000 + (5-3) × 20,000 = 80,000원
```

### ⚠️ 접지 제한사항

```
┌─────────────────────────────────────────┐
│  A5 이하는 일반접지 불가!                │
│                                         │
│  A6, A5 → ❌ 일반접지 불가               │
│           → 미니접지 별도 (전화요망)     │
│  A4 이상 → ✅ 일반접지 가능              │
└─────────────────────────────────────────┘
```

| 규격 | 일반접지 | 비고 |
|------|---------|------|
| A6 | ❌ 불가 | 미니접지 전화요망 |
| A5 | ❌ 불가 | 미니접지 전화요망 |
| A4 | ✅ 가능 | |
| A3 | ✅ 가능 | |
| A2 | ✅ 가능 | |

## 📋 재단비 (1연당)

### 가격표

| 등분 | 가격 | 비고 |
|------|------|------|
| 2등분 | 10,000원 | |
| 3등분 | 10,000원 | |
| 4등분 이상 | 전화요망 | 별도 견적 |

### 재단 예시

```
2등분: A3 → A4 2장
3등분: A3 → 세로로 3등분
4등분 이상: 복잡한 재단 → 전화요망
```

---

## 🧮 후가공비 계산 공식

### 코팅비
```
코팅비 = 연수 × 코팅단가 × 면수(1 or 2)

조건: 150g 이상만 가능
```

### 접지비
```
3단 이하: 연수 × 40,000원
4단 이상: 연수 × (40,000 + (단수-3) × 20,000)원

조건: A4 이상만 일반접지 가능
```

### 재단비
```
2~3등분: 연수 × 10,000원
4등분 이상: 전화요망
```

---

## 📝 계산 예시

### 예시 1: 150g A4 전단지 4,000장 양면 유광코팅
```
A4 = 8절 → 1연당 4,000장
4,000장 ÷ 4,000장 = 1연

코팅비 = 1연 × 80,000원 × 2(양면) = 160,000원
```

### 예시 2: 180g A3 리플렛 2,000장 3단접지
```
A3 = 4절 → 1연당 2,000장
2,000장 ÷ 2,000장 = 1연

접지비 = 1연 × 40,000원 = 40,000원
```

### 예시 3: A4 8,000장 2등분 재단
```
A4 = 8절 → 1연당 4,000장
8,000장 ÷ 4,000장 = 2연

재단비 = 2연 × 10,000원 = 20,000원
```
