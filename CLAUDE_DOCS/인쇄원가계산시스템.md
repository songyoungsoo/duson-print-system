# 인쇄원가계산시스템

## 📐 총원가 공식

```
총원가 = 종이값 + 판비 + 인쇄비 + 후가공비
```

---

## 🔢 1단계: 종이값

### 기본 공식
```
종이값 = 지류기준가 × (평량÷90) × (B시리즈면 1.5) × 연수
```

### 지류별 기준가 (90g A시리즈 1연)

| 지류 | 가격 |
|------|------|
| At (아트지) | 35,000원 |
| sw (스노우지) | 35,000원 |
| 몽블랑 | 80,000원 |

### 평량별 가격 (A시리즈 1연)

| 평량 | 가격 | 배수 |
|------|------|------|
| 90g | 35,000원 | ×1.00 |
| 120g | 46,650원 | ×1.33 |
| 150g | 58,333원 | ×1.67 |
| 180g | 70,000원 | ×2.00 |
| 200g | 77,778원 | ×2.22 |
| 250g | 97,222원 | ×2.78 |
| 300g | 116,667원 | ×3.33 |

### B시리즈 = A시리즈 × 1.5

| 평량 | A시리즈 | B시리즈 |
|------|---------|---------|
| 90g | 35,000원 | 52,500원 |
| 150g | 58,333원 | 87,500원 |

### 절수별 1연당 매수

| 규격 | 절수 | 1연당 매수 |
|------|------|-----------|
| A2 | 2절 | 1,000장 |
| A3 | 4절 | 2,000장 |
| A4 | 8절 | 4,000장 |
| A5 | 16절 | 8,000장 |
| A6 | 32절 | 16,000장 |

---

## 🔢 2단계: 판비

### 합판 (90g 전용)
```
A2 32,000원 기준 등분 (최소 4등분)
양면 = × 2
```

| 규격 | 등분 | 4도 단면 | 4도 양면 |
|------|------|----------|----------|
| A2 | 1 | 32,000원 | 64,000원 |
| A3 | 2 | 16,000원 | 32,000원 |
| A4~A6 | 4 | 8,000원 | 16,000원 |

### 독판 (120g 이상)
```
규격별 가격, 양면 = × 2
```

| 규격 | 4도 단면 | 4도 양면 |
|------|----------|----------|
| A2 | 32,000원 | 64,000원 |
| A3 | 22,000원 | 44,000원 |
| A4~A6 | 12,000원 | 24,000원 |

---

## 🔢 3단계: 인쇄비

### 합판 (90g) - 1연당

| 인쇄 방식 | 가격 |
|----------|------|
| 단면 4도 | 10,000원 |
| 양면 4도+4도 | 15,000원 (×1.5) |

### 독판 (120g+) - 1연당

| 인쇄 방식 | 가격 |
|----------|------|
| 단면 4도 | 40,000원 |
| 양면 4도+4도 | 60,000원 (×1.5) |

※ A/B시리즈 인쇄비 동일 (관행)
※ 양면 = 1.5배 (8색기 효율)

---

## 🔢 4단계: 후가공비

### 코팅 (1연당, 150g 이상만)

| 종류 | 단면 | 양면 |
|------|------|------|
| 유광 | 80,000원 | 160,000원 |
| 무광 | 90,000원 | 180,000원 |

| 단수 | 가격 |
|------|------|
| 3단 이하 | 40,000원 |

| n단 | +20,000원씩 |

### 재단 (1연당)

|------|------|

## ⚠️ 주의사항

### 합판 vs 독판
- **90g만 합판** (공동 인쇄, 저렴)
- **120g 이상 독판** (단독 인쇄, 비쌈)

### 코팅 제한
- **150g 이상만 가능** (얇으면 말림)

### 접지 제한
- **A4 이상만 일반접지**
- A5 이하 미니접지 별도

### A/B시리즈
- 종이값: B = A × 1.5
- 판비: 합판 동일, 독판 B = A × 1.5
- 인쇄비: **동일** (관행)

### 양면 인쇄
- 판비: × 2
- 인쇄비: × 1.5 (8색기 효율)
