# 인쇄원가계산 1 - 종이값 계산

## 📐 기본 개념

### 연(聯) 시스템
- **1연 = 전지(全紙) 500장**
- 전지를 절수(折數)로 자르면 그만큼 매수가 늘어남
- **1연당 매수 = 500 × 절수**

## 🧮 종이값 계산 공식

### 5단계 계산법

```
① 전지 1장당 몇 매? = (전지가로÷제품가로) × (전지세로÷제품세로)
② 1연당 매수 = 500장 × ①
③ 주문수량 ÷ ② = 필요 연수
④ 지류 기준가 × (평량÷90) × (B시리즈면 1.5) = 1연 가격
⑤ ③ × ④ = 종이값
```

### 공식 요약

```
종이값 = [지류 기준가] × (평량 ÷ 90) × (B시리즈면 1.5) × 필요연수
```

## 📝 계산 예시

### 예시 1: 90g A4 8,000장
```
A4 = 8절 → 1연당 4,000장
8,000장 ÷ 4,000장 = 2연
2연 × 35,000원 = 70,000원
```

### 예시 2: 90g A6 8,000매
```
A6 = 32절 → 1연당 16,000장
8,000장 ÷ 16,000장 = 0.5연
0.5연 × 35,000원 = 17,500원
```

### 예시 3: 명함 250g sw 90×50mm 500매
```
① 전지당 매수: (594÷90) × (841÷50) = 6 × 16 = 96매
② 1연당 매수: 500 × 96 = 48,000매
③ 필요 연수: 500 ÷ 48,000 = 0.0104연
④ 250g sw 1연: 35,000 × (250÷90) = 97,222원
⑤ 종이값: 97,222 × 0.0104 = 약 1,000원
```
