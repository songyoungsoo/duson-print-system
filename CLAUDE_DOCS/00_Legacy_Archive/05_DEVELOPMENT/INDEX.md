# 💻 05_DEVELOPMENT - 개발 가이드

## 📋 요약
프론트엔드 UI/UX, 트러블슈팅, 최적화

### 🎨 프론트엔드
- **CSS**: 통합 디자인 시스템
- **JS**: 실시간 계산 + 인터랙션
- **반응형**: 모바일 우선 설계

### 🐛 트러블슈팅
- **일반 오류**: DB 연결, 파일 권한
- **계산 오류**: JavaScript 함수 검증
- **업로드 오류**: 파일 크기, 확장자

### ⚡ 성능 최적화
- **CSS**: 인라인 → 외부 파일
- **JS**: 모듈화 + 압축
- **이미지**: 최적화 + lazy loading

### 🔧 개발 도구
- **디버그**: 브라우저 개발자 도구
- **로그**: PHP error.log
- **테스트**: 수동 + 자동화

### 🔌 MCP Integration
- **MCP 설치**: mcp-installer 사용
- **설정 관리**: User/Project 스코프
- **검증**: claude --debug + /mcp
