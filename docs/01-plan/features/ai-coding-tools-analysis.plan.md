# Plan: AI 코딩 도구 시장 경쟁사 분석

> **Feature**: ai-coding-tools-analysis
> **Created**: 2026-02-05
> **Phase**: Plan
> **Output**: PPT 슬라이드용 마크다운 콘텐츠

---

## 1. 목표 (Objective)

AI 코딩 도구 시장의 주요 경쟁사(Cursor, Augment, Claude Code)를 비교 분석하여
프레젠테이션 슬라이드 제작에 활용할 수 있는 구조화된 콘텐츠를 생성한다.

### 1.1 핵심 질문
- 각 도구의 핵심 차별점은 무엇인가?
- 타겟 사용자 및 사용 시나리오는?
- 가격 정책 및 비즈니스 모델 차이는?
- 기술적 강점과 한계는?

### 1.2 예상 산출물
- **Design 문서**: 슬라이드 구성 및 콘텐츠 상세
- **최종 Output**: `ai-coding-tools-slides.md` (PPT 변환용)

---

## 2. 분석 대상 (Scope)

### 2.1 주요 경쟁사 3사

| 제품 | 개발사 | 카테고리 |
|------|--------|----------|
| **Cursor** | Anysphere | AI-Native IDE |
| **Augment** | Augment (구 Codeium) | AI Coding Assistant |
| **Claude Code** | Anthropic | AI CLI Tool |

### 2.2 비교 항목

1. **제품 개요**: 출시일, 버전, 주요 특징
2. **기술 스택**: 사용 AI 모델, 아키텍처
3. **주요 기능**: 코드 생성, 리팩토링, 디버깅, 멀티파일 편집
4. **가격 정책**: 무료/유료, 구독 모델
5. **타겟 시장**: 개인/팀/기업, 사용 규모
6. **강점/약점**: SWOT 요약
7. **시장 포지셔닝**: 차별화 전략

---

## 3. 슬라이드 구성안 (Preliminary Structure)

```
슬라이드 1: 타이틀 - AI 코딩 도구 시장 경쟁사 분석
슬라이드 2: 시장 개요 - AI 코딩 도구 시장 현황
슬라이드 3: 경쟁사 소개 - 3사 간략 소개
슬라이드 4-6: 개별 분석 - Cursor / Augment / Claude Code
슬라이드 7: 비교표 - 기능/가격/타겟 종합 비교
슬라이드 8: SWOT 분석 - 각 제품별 강점/약점
슬라이드 9: 시장 포지셔닝 맵
슬라이드 10: 결론 및 시사점
```
