## 2. 리서치 결과 요약

### 2.1 Cursor
- **유형**: AI-Native IDE (VS Code 포크)
- **특징**: 전체 코드베이스 이해, 다중 AI 모델 지원
- **가격**: Free / Pro $20 / Pro+ $60 / Ultra $200 / Teams $40

### 2.2 Augment
- **유형**: AI Coding Assistant (IDE 플러그인)
- **특징**: Context Engine, 100+ 네이티브 도구, MCP 지원
- **가격**: Community 무료 / Developer $50 / Pro $100 / Max $250

### 2.3 Claude Code
- **유형**: AI CLI Tool (터미널 기반)
- **특징**: 에이전틱 코딩, Git 워크플로우, IDE 확장 지원
- **가격**: Pro $20 / Max $100 / Max $200
