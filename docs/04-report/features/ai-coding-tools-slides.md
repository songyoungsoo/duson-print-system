# 시장 개요

## AI 코딩 도구 시장 현황

- **시장 규모**: 2026년 AI 코딩 도구 시장 급성장
- **주요 트렌드**:
  - IDE 통합에서 **에이전틱(Agentic) 코딩**으로 진화
  - 단순 자동완성 → **전체 코드베이스 이해**
  - 단일 모델 → **멀티 AI 모델** 지원

### 핵심 경쟁 요소
| 요소 | 설명 |
|------|------|
| 컨텍스트 이해 | 프로젝트 전체 구조 파악 능력 |
| 에이전트 기능 | 멀티파일 자동 수정 |
| 가격 효율성 | 토큰/크레딧 사용량 대비 가치 |

---

# 경쟁사 소개

## 3대 AI 코딩 도구

| 제품 | 개발사 | 유형 | 출시 |
|------|--------|------|------|
| **Cursor** | Anysphere | AI-Native IDE | 2023 |
| **Augment** | Augment Code | AI Assistant | 2023 |
| **Claude Code** | Anthropic | AI CLI Tool | 2025 |

### 포지셔닝 차이
- **Cursor**: VS Code 대체, 완전한 IDE 경험
- **Augment**: 기존 IDE에 플러그인으로 추가
- **Claude Code**: 터미널 중심, IDE 연동 가능

---

# Cursor 분석

## AI-Native IDE의 선두주자

### 핵심 특징
- VS Code 포크 기반 - **기존 익스텐션 100% 호환**
- **전체 코드베이스 임베딩** - 50,000줄 이상 프로젝트 이해
- **멀티 AI 모델**: GPT-5, Claude, Gemini 선택 가능
- **Tab 자동완성**: 항상 작동하는 AI 자동완성
- **Agent 모드**: 복잡한 멀티파일 리팩토링

### 가격 정책
| 플랜 | 가격 | 특징 |
|------|------|------|
| Hobby | 무료 | 2,000 완성/월, 50 슬로우 요청 |
| Pro | $20/월 | 무제한 Tab, $20 크레딧 |
| Pro+ | $60/월 | 3배 크레딧 |
| Ultra | $200/월 | 20배 크레딧, 신기능 조기 접근 |
| Teams | $40/유저 | SSO, 팀 관리 |

---

# Augment 분석

## 엔터프라이즈 코딩 어시스턴트

### 핵심 특징
- **Context Engine**: 전체 스택 실시간 이해
- **100+ 네이티브 도구** + MCP 통합
- **IDE 독립적**: VS Code, JetBrains, Vim 지원
- **AI 코드 리뷰**: GitHub PR 자동 분석
- **상용 TOS**: AI 학습에 코드 사용 안 함

### 가격 정책
| 플랜 | 가격 | 메시지/월 |
|------|------|-----------|
| Community | 무료 | 50 |
| Developer | $50/월 | 600 |
| Pro | $100/월 | 1,500 |
| Max | $250/월 | 4,500 |

> **크레딧 예시**: 소규모 작업(10 도구호출) ~300 크레딧, 복잡한 작업(60 도구호출) ~4,300 크레딧

---

# Claude Code 분석

## 터미널 기반 에이전틱 코딩

### 핵심 특징
- **터미널 네이티브**: 개발자 워크플로우에 자연스럽게 통합
- **코드베이스 이해**: 전체 프로젝트 구조 파악
- **Git 워크플로우**: 커밋, PR 생성 자동화
- **자연어 명령**: 복잡한 작업을 대화로 처리
- **IDE 확장**: VS Code, Cursor, JetBrains 연동

### 가격 정책 (Claude 구독 포함)
| 플랜 | 가격 | 특징 |
|------|------|------|
| Pro | $20/월 | 기본 Claude Code 접근 |
| Max $100 | $100/월 | 5x 사용량, Opus 접근 |
| Max $200 | $200/월 | 20x 사용량, 무제한 Opus |

> **2026 신기능**: PDF 페이지 지정 읽기, --from-pr 플래그, 3배 메모리 최적화

# 결론 및 선택 가이드

## 사용자별 추천

| 사용자 유형 | 추천 도구 | 이유 |
|------------|----------|------|
| **VS Code 사용자** | Cursor | 가장 자연스러운 전환 |
| **JetBrains 사용자** | Augment | IDE 독립적 |
| **터미널 파워유저** | Claude Code | CLI 네이티브 |
| **팀 협업 중심** | Cursor Teams | SSO, 관리도구 |
| **보안 민감 기업** | Augment | AI 학습 배제 |

## 핵심 시사점

1. **IDE 대체 vs 보조**: Cursor는 대체, 나머지는 보조
2. **가격 전략 다변화**: 무료→프리미엄 모두 존재
3. **에이전틱 코딩 표준화**: 3사 모두 에이전트 모드 지원
4. **컨텍스트가 핵심**: 전체 코드베이스 이해 능력이 차별점
